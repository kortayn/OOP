Задание 1:
Создать класс IntArray, реализующий одномерный динамический массив.
Сам массив хранить в поле типа unique_prt<int>, также хранить в отдельном поле длину массива.
В классе должно быть два конструктора: конструктор поумолчанию создаёт пустой массив стандартной длины, второй конструктор должен принимать размер, создаваемого массива.
Для доступа к элементам реализовать два метода int get(size_t index), void set(size_t index, int value).
Реализовать метод int length(), возвращающий длину массива.
Реализовать статический метод random_fill, принимающий ссылку на массив и ссылку на распределение uniform_int_distribution, и заполняющий массив случайными числами.
Переопределить оператор <<. Он должен возвращать поток в котором находится содержимое массива.

Задание 2:
Реализовать класс IntArray2D, реализующий двумерный прямоугольный динамический массив, используя класс из предыдущего задания.
В классе должно быть три поля - поле с массивом и два поля с количесвом строк и столбцов.
Реализовать int columns(), int rows(), int get(size_t row, size_t col), void set(size_t row, size_t col, int value).
Реализовать статический метод random_fill, принимающий ссылку на массив и ссылку на распределение uniform_int_distribution, и заполняющий массив случайными числами.
Переопределить оператор <<. Он должен возвращать поток в котором находится содержимое массива.
